Iniciar sesion

¿Como iniciar sesion?


Para iniciar sesion es necesario tener una cuenta en caso de no tener una dirijase a registro

Una vez disponga de una cuenta deberas de introducir los datos con los que creaste la cuenta 



En este apartado deberas de entroducir el email o usuario con el que creaste la cuenta en caso de no ser correcto no te dejara iniciar sesion



En este apartado deberas de introducir la contraseña que pusiste al crear la cuenta al igual que con el usuario en caso de no ser correcto el inicio de sesion fallara 



Para iniciar sesion deberas de darle al boton de iniciar sesion



Este boton permite guardar sesion esto sirve para que la proxima vez que abras la aplicacion no tengas que introducir nuevamente tus credenciales en caso de que lo hayas seleccionado y no quieras que siga iniciando sesion deberas de ir a Ajustes de usuario
