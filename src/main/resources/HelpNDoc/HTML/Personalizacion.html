Personalización

¿Que es el apartado de personalizacion?

El apartado de personalizacion permite al usuario cambiar diferentes apartados visuales a gusto del usuario



Actualemente el unico apartado con funcionalidad es el de temas



El apartado de temas permite cambiar la paleta de colores de la aplicacion actualmente la paleta negra no esta disponible una vez le des al tema debes reiniciar la aplicacion para que se apliquen los cambios.

Para poner un tema personalizado le damos a personalizado y se abrira una vetana con diferentes opciones donde seleccionar el color de los diferentes apartados de la aplicacion.
